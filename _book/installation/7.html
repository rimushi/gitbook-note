原型模式

// 原型模式 （Prototype pattern）：通俗点讲就是创建一个
// 共享的原型，并通过拷贝这些原型创建新的对象。用于创建
// 重复的对象，这种类型的设计模式属于创建型模式

// 因为不是构造函数，所以不用大写
var someCar = {
    drive: function () { },
    name: '马自达 3'
};
// 使用Object.create创建一个新车x
var anotherCar = Object.create(someCar);
anotherCar.name = '丰田佳美';
